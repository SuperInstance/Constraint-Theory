How It Works

The derivative is computed using the central difference method, which provides O(h^2) truncation error. The slope is then encoded as a 12-bit dodecet for geometric precision.

Move your mouse over the canvas to see the tangent line at different points. The green dot shows the current point, and the magenta dashed line shows the tangent.

How It Works

How It Works

The gradient of a scalar field f(x,y) points in the direction of steepest increase. Green particles follow gradient descent, flowing toward local minima like water flowing downhill.

Each gradient vector is encoded as two dodecets (one for each component), providing 16x more precision than 8-bit encoding.

Code Example

// Differentiation with dodecet encoding
const { value, encodedSlope, error } = DodecetDifferentiation.derivative(
    (x) => Math.sin(x),
    Math.PI / 4
);

console.log(`Derivative: ${value}`);        // cos(PI/4) = 0.707
console.log(`Encoded: ${encodedSlope.toHexString()}`);  // "59E"
console.log(`Error: ${error}`);             // ~0.000001

// Integration with Simpson's rule
const { value, encodedArea, strips } = DodecetIntegration.integrate(
    (x) => Math.sin(x),
    0, Math.PI
);

console.log(`Integral: ${value}`);          // 2.0
console.log(`Encoded: ${encodedArea.toHexString()}`);   // "FFF"

// Gradient computation
const { grad, partials, magnitude } = DodecetGradient.gradient(
    (x, y, z) => x*x + y*y + z*z,
    { x: 1, y: 2, z: 3 }
);

console.log(`Gradient: (${partials.dx}, ${partials.dy}, ${partials.dz})`);
// (2, 4, 6) - gradient of f = x^2 + y^2 + z^2
